What are Meta Tags?
Meta tags are used to provide additional information to the browsers. This information will be available to the browser and not displayed on the rendered page. The information contained within the Meta tags may range from a brief description about the site content, keywords, author information to additional browser specific information.  
General Syntax of Meta Tag
<meta name=”name” content=”content” />
Another most popular syntax for Meta Tag is:
<meta http-equiv=”attribute-value” content=”value” />

Different types of Meta Tags:
1.     Http-Equiv attribute:
<meta http-equiv="content-type" content="text/html;charset=UTF-8" />
Http-Equiv is a very important attribute. They control the action of browsers Information conveyed by the this tag includes the character set used, refresh information etc. There is lot of action that can be performed with them. As an example, let me show how to redirect the page to another page after 10 sec the user sees the page.
<html>
     <head>
          <meta http-equiv="Refresh" Content="10; URL=http://www.google.co.in" />
     </head>
     <body>
          Please wait... Redirecting to Google home page.
     </body>
</html>
There are many other attributes, which combined with Http-Equiv, can make them more meaningful for the browsers to interpret. Some information is shared below in important references section.
2.     Description attribute:
<meta name="description" content="Free Web tutorials" />
The Description attribute provides a brief description about the content of the page. There is no length restriction imposed on the content. But search engine like Google usually ignore content over 150 characters. So do try to keep the description to less than 150 characters.
3.     Keywords attribute:
<meta name="keywords" content="HTML,CSS,XML,JavaScript" />
Keyword attribute is used by search engines to associate a web page with some keyword. However most of the major search engines like Google, etc. have stopped using description for scanning keywords due to its misuse to gain undue popularity.
4.     Author attribute:
<meta name="author" content="Abhishek Gopalan" />
Author attribute provides information about the author of the page.
Importance of Meta Tags:
Meta tags provide some additional information about the page to the web browser. They are optional and need not be present in the web page. Meta tags gain prominence when search engines are involved. Search engines usually pick up key information from the Meta information present in the page. So if you want your web page to be more popular over the search engines, place some meta information in the page.
